# Hanoi-MIPS
- Implement Hanoi tower in MIPS. 
- Use QtSpim to execute MIPS instructions in x86 CPUs.
  - **Reinitialize Simulator** before any execution.
  - *Single step* is your good friend for debugging.

## Quest
- Enter n which greater than 0 and smaller than 32.
- Return the steps needed for a Hanoi tower to move n pieces from a to c.
